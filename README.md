## 简介

此仓库主要是记录下自己学习 Vue、Vuex 相关库源码的笔记和注释，方便以后忘记可以从学习轨迹中迅速上手。

## 目录结构

- vue-src
- mvvm-simple-diy
- vuex-src
- vue-router-src
